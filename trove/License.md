# License

This dataset contains metadata records collected from Trove, the digital
discovery platform of the National Library of Australia.

All rights to the original materials and their associated metadata
belong to the National Library of Australia and the contributing
institutions represented in Trove.

The dataset distributed in this repository includes only metadata fields
describing newspaper records. The original articles remain accessible
through the Trove platform.

The metadata was programmatically collected via the Trove API for the
purposes of research and open data initiatives.

This dataset was compiled for the Open Data Armenia in order
to identify, collect, and preserve references to Armenian cultural
heritage and historical events in international archives and libraries.

Users of this dataset should acknowledge:

Trove -- National Library of Australia

Links to the original Trove records are provided in the dataset so that
users can access the authoritative sources directly.
